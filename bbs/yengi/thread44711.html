2016-يىللىق ھۆكۈمەت تۈر ياردەم مەبلىغى بىلەن تەمىنلەيدىغان كەسىپلەر ۋە ياردەم
مەبلىغىنىڭ پۇل سوممىسى

شانئال مەكتىپىدىن كامران تەرجىمىسى


2016-يىلى قىلىۋاتقان كەسپىڭىزگە دۆلەتنىڭ ياردەم مەبلىغىنى قولغا كەلتۈرمەكچى بولسىڭىز ھېچ بولمىغاندا قانداق تۈرلەرگە ياردەم مەبلىغى بېرىلىدىغانلىقىنى بىلىشىڭىز كېرەك.
1. يېزائىگىلىك مىنىستىرلىقى
تېرىقچىلىق ،باقمىچىلىق كەسىپلىرى
1) نەسىللىك چارۋا ،ئۆي قۇشلىرى فېرمىسى تۈرىنى ئىلتىماسقىلىش؛
2) ئەلا سۈپەتلىك تاۋارئاشلىق بازىسى تۈرىنى ئىلتىماس قىلىش؛
3)ئەلا سۈپەتلىك  شالغۇت ئاشلىق بازىسى تۈرىنى ئىلتىماس قىلىش؛
4) ئەلا سۈپەتلىك  سۇ مەھسۇلاتلىرى بازىسى تۈرىنى ئىلتىماس قىلىش؛
5) ئەلا سۈپەتلىك  كۆكتات( يېيىلىدىغان زەمبۇرۇغ ) بازىسى تۈرىنى ئىلتىماس قىلىش؛
6) ئەلا سۈپەتلىك  گۈل-گىياھ بازىسى تۈرىنى ئىلتىماس قىلىش؛
7)ئۆلچەملەشكەن چارۋا-ئۆيقۇشلىرى باقمىچىلىق رايونى سىناق نوختا بازىسى تۈرى ؛
8) ئەلا سۈپەتلىك  مېۋە-چىۋە ئۈلگە كۆرسىتىش بازىسى تۈرىنى ئىلتىماس قىلىش؛
9) ئەلا سۈپەتلىك  پىلە بېقىش ئۈلگە كۆرسىتىش بازىسى تۈرىنى ئىلتىماس قىلىش؛
10)زىرائەت غول-شاخلىرىبىلەن چارۋا بېقىش تۈرىنى ئىلتىماس قىلىش؛
11)پاتقاق گازى تۈرىنى ئىلتىماس قىلىش؛
12)ئىقتىسادى ئورمان بازىسىتۈرىنى ئىلتىماس قىلىش؛
13)ئەسلىھەلىك يېزا ئىگىلىكتۈرىنى ئىلتىماس قىلىش؛
14)يۇقىرى ھوسۇل ئالدىغان زىرائەتلەردە بۆسۈش ھاسىل قىلىش تۈرىگە ئىلتىماس قىلىش؛

پىششىقلاپ ئىشلەش كەسىپلىرى
1)ئاشلىق پىششىقلاپ ئىشلەشتۈرىگە ئىلتىماس قىلىش؛
2)مېۋە-چىۋە پىششىقلاپ ئىشلەشتۈرىگە ئىلتىماس قىلىش؛
3)چارۋىچىلىق مەھسۇلاتلىرىنى پىششىقلاپ ئىشلەش تۈرىگە ئىلتىماس قىلىش؛
4)مايلىق دان تۈرىدىكىلەرنى پىششىقلاپ ئىشلەش تۈرىگە ئىلتىماس قىلىش؛
5)سۇ مەھسۇلاتلىرىنى پىششىقلاپ ئىشلەش تۈرىگە ئىلتىماس قىلىش؛
6)بىيو-تېخنىكا مەھسۇلاتلىرىنى پىششىقلاپ ئىشلەش تۈرىگە ئىلتىماس قىلىش؛
7) يۇقىرى تېخنىكىلىق يېزائىگىلىك مەھسۇلاتلىرىنى پىششىقلاپ ئىشلەش تۈرىگە ئىلتىماس قىلىش؛
8)يېمەكلىك پىششىقلاپ ئىشلەشتۈرىگە ئىلتىماس قىلىش؛
9)سۇيۇق ھالەتتىكى سۈت مەھسۇلاتلىرىنى پىششىقلاپ ئىشلەش تۈرىگە ئىلتىماس قىلىش؛

مال ئوبوروت ئەسلىھەلىرى كەسپى
1) يېڭى پېتى ساقلاش ئەسلىھەلىرى قۇرۇلۇشى تۈرىگە ئىلتىماس قىلىش؛
2) مال توپ تارقىتىش بازىرى قۇرۇلۇشى تۈرىگە ئىلتىماس قىلىش؛

ئاساسى مۇئەسسەسەقۇرۇلۇشى
1) ئوتتۇرا-تۆۋەن ھوسۇللۇق ئېتىزلارنى ئۆزگەرتىش تۈرىگەئىلتىماس قىلىش؛
2) ئېتىزلاردىكى تۇك،سۇ يوللىرىنى ئۆزگەرتىش قۇرۇلۇشىتۈرىگە ئىلتىماس قىلىش؛
3) سۇ تېجەپ سۇغۇرۇش تۈرىگە ئىلتىماس قىلىش؛
4) ئوتتۇرا-كىچىك دائىرىدىكى سۇغۇرۇش سۇ ئامبارلىرى تۈرىگەئىلتىماس قىلىش؛
5) شورلۇق يەرلەرنى ئۆزگەرتىش تۈرىگە ئىلتىماس قىلىش؛
6) قۇملۇق چۆللەرنى ئۆزگەرتىش تۈرىگە ئىلتىماس قىلىش؛
7) ئوتلاق قۇرۇلۇشى تۈرىگە ئىلتىماس قىلىش؛

تەرەققىيات ئىسلاھات كومىتېتى
1) دۆلەت يۇقىرى تېخنىكا كەسىپلىرى تەرەققىيات تۈرى ۋەيۇقىرى تېخنىكىلىق زامانىۋى يېزا ئىگىلىك قۇرۇلۇشى تۈرىگە ئىلتىماس قىلىش؛
2) يېزا ئىگىلىكىنى كەسىپلەشتۈرۈش باشلامچى كارخانا تۈرىگەئىلتىماس قىلىش؛
3) زامانىۋى مىللىي تېبابەت كەسىپلىرى تۈرىگە ئىلتىماسقىلىش؛

مالىيە نازارىتى
1) دۆلەت يېزا ئىگىلىكىنى  ئۇنىۋېرسال ئېچىش تۈرىگە ئىلتىماس قىلىش؛
2) دۆلەت يېزا ئىگىلىكىنى  ئۇنىۋېرسال ئېچىشنى كەسىپلەشتۈرۈش تۈرىگە ئىلتىماسقىلىش؛
3) جۇڭيى تېبابىتى دورىلىرى ماتېرىياللىرىنى ئىشلەپ چىقىرىشنىيۆلەش تۈرىگە ئىلتىماس قىلىش؛
4) دۆلەت يېزا ئىگىلىكىنى ئۇنىۋېرسال ئېچىشنى كەسىپلەشتۈرۈشمەخسۇس تۈرىگە ئىلتىماس قىلىش؛

سودا مىنىستىرلىقى
1) " ئونمىڭ كەنت مىڭ يېزا " دا بازار قۇرۇشسىناق نۇقتا تۈرىگە ئىلتىماس قىلىش؛

پەن-تېخنىكامىنىستىرلىقى
1) پەن-تېخنىكىلىق ئوتتۇرا-كىچىك كارخانىلارنىڭ تېخنىكىدايېڭىلىق يارىتىش فوندى تۈرىگە ئىلتىماس قىلىش؛
2) يېزا-ئىگىلىك تەتقىقات نەتىجىسىنى ئەمەلىيەتتە قوللىنىشمەبلىغى تۈرىگە ئىلتىماس قىلىش؛
3) دۆلەت كېڭەيتىش پىلانى تۈرىگە ئىلتىماس قىلىش؛
4) دۆلەت يۇقىرى تېخنىكا پەن-تەتقىقات تۈرىگە ئىلتىماسقىلىش؛

دۆلەتلىك دەرىجىلىك تۈرلەرنىڭ ياردەم پۇلى قانچىلىك؟
مالىيە مىنىستىرلىقى قىسمەن تۈرلەرنىڭ پۇل سوممىسى
1. باشلامچى كارخانىلارنىڭ كەسىپلەر تەرەققىياتىنى ئىلگىرى سۈرۈش ۋە " بىر ناھىيەدە بىر ئالاھىدە كەسىپ بۇلۇش" سىناق نوختا تۈرى( 3مىليوندىن---8 مىليونغىچە )
2. يېزا ئىگىلىكىنى ئۇنىۋېرسال ئېچىش يېزا ئىگىلىكىنى كەسىپلەشتۈرۈش تۈرى ( 500مىڭدىن---3مىليونغىچە )
3. يېزا ئىگىلىكىنى ئۇنىۋېرسال ئېچىش ئورمانچىلىق مەخسۇس تۈرى ( 1مىليون200مىڭدىن تۆۋەن ئەمەس)
4. يېزا ئىگىلىكىنى ئۇنىۋېرسال ئېچىش يېڭىچە ھەمكارلىق ئۈلگە كۆرسىتىش تۈرى ( 500مىڭدىن---2مىليونغىچە)
5. يېزا ئىگىلىكىنى ئۇنىۋېرسال ئېچىش تۇپرىقىنى تۈزەش تۈرى ( 5مىليون ئەتراپىدا )
6. باغچىلاشقان زامانىۋى يېزائىگىلىك تۈرىنى ئىلتىماسقىلىش ( 10مىليوندىن---20مىليونغىچە)
7. ئوتتۇرا كۆلەملىك سۇغۇرۇش رايونىنى سۇ تېجەش بۇيىچە ئۆزگەرتىش تۈرى ( 10مىليوندىن كەم ئەمەس )

يېزا ئىگىلىك مىنىستىرلىقى قىسمەن تۈرلەرنىڭ پۇل سوممىسى
1. " كۆكتات سېۋىتى " قۇرۇلۇشى مەھسۇلاتلىرىنىئىشلەپچىقىرىش تۈرى ( 3مىليون ئىچىدە )
2. يېزا ئىگىلىكىنى ئۇنىۋېرسال ئېچىش يېزائىگىلىك مىنىستىرلىقىمەخسۇس تۈرى( ئەلا نەسىلنى كۆپەيتىش،ئالاھىدە كوزىر تېرىقچىلىق،باقمىچىلىق تۈرى 2مىليوندىن---5مىليونغىچە)
3. چوڭ،ئوتتۇرا تىپتىكى پاتقاق گازى قۇرۇلۇشى مەركەزبىۋاسىتە مەبلەغ سېلىش تۈرى ( 1مىليوندىن---2مىليونغىچە )
4. يېزائىگىلىك مەھسۇلاتلىرىنى چىققان يېرىدە تۆۋەن دەرىجىدەپىششىقلاپ ئىشلەش تولۇقلىما تۈرى ( ئومۇمىي سېلىنغان مەبلەغنىڭ %30 ئەتراپىدا )
تەرەققىياتئىسلاھات  كومىتېتى قىسمەن تۈرلىرىنىڭ پۇل سوممىسى
1. زامانىۋى يېزا ئىگىلىكئۈلگە كۆرسىتىش تۈرى ( 2مىليوندىن-200مىليونغىچە )
2. نوختىلىق كەسىپلەرنى جانلاندۇرۇش ۋە تېخنىكىلىق ئۆزگەرتىشمەخسۇس تۈرى ( ئومۇمىي سېلىنغان مەبلەغنىڭ %10 )
3. بايلىقنى تېجەش مۇھىت قوغداش مەركەز خام چوتى تۈرى(ئومۇمىي سېلىنغان مەبلەغنىڭ %10)
4. ئىقتىساد-سودا ساھەسىدىكى مەركەز مەبلەغ تۈرى ( 5مىليون ئەتراپىدا )
5. قوي ئۆلچەملىك باقمىچىلىق قوتىنى ( رايونى ) تۈرى( 300مىڭدىن---1مىليونغىچە)
6. سۈت كالىسىنى ئۆلچەملىك بېقىش فېرمىسى ( رايونى )تۈرى ( 800مىڭدىن1مىليون700مىڭ )
7. بىيولوگىيەلىك ئېنېرگىيەدىن ئۇنىۋېرسال پايدىلىنىش ئۈلگە كۆرسىتىش تۈرى ( سېلىنغان ئومۇمىي مەبلەغنىڭ %10 ئەتراپىدا)

باشقا تۈرلەرنىڭ پۇل سوممىسى
1. پەن-تېخنىكا مىنىستىرلىقى : يېزائىگىلىك پەن-تەتقىقاتنەتىجىلىرىنى ئەمەلىيەتتە قوللىنىش تۈرى ( 600مىڭدىن---3مىليونغىچە )
2. پەن-تېخنىكا مىنىستىرلىقى : ئوتتۇرا-كىچىك كارخانىلارتېخنىكىدا يېڭىلىق يارىتىش فوندىنىڭ زامانىۋى يېزائىگىلىك ساھەسى تۈرى ( 800مىڭدىن ئاشمايدۇ )
3. سانائەت ۋە ئۇچۇرلاشتۇرۇش مىنىستىرلىقى : ئوتتۇرا-كىچىك كارخانىلارنىڭ تەرەققىياتى مەخسۇس تۈرى ( 3مىليون )
4. نامراتلارنى يۆلەش ئىشخانىسى : كەسىپلەشتۈرۈپ نامراتلارنى يۆلەش تۈرى ( 5 مىليون )

بۇندىن كېيىن تۈر مەبلەغلىرى " كەسىپلەر زەنجىرى " نى بەرپا قىلىشقا ئاغىدۇ
2014-يىلىدىن باشلاپ دۆلەت" پىششىقلاپ ئىشلەش كوپىراتىپى " ئۈلگە كۆرسىتىش تۈرىنى مەخسۇس ئوتتۇرىغاقويدى،پىلان بويىچە 2014-يىلىدىن-2016-يىلىغىچە دۆلەت دەرىجىلىك 1000 " پىششىقلاپئىشلەش كوپىراتىپى "نى قۇرۇپ چىقتى. تۈر مەبلىغى ئاستا-ئاستا يۇقىرى ئۆلچەملىك ئەسلىھەلىك يېزا ئىگىلىكى،كۆلەملەشكەن تېرىقچىلىق ،باقمىچىلىقتىن،مەھسۇلاتلارنى ئوراپقاچىلاش،پىششىقلاپ ئىشلەش،ئوبوروت قاتارلىق ساھەلەرگە ئاغدى. يېزائىگىلىك كارخانىلىرىدېھقانلار ،دېھقانلار كوپىراتىپى بىلەن مەنپەئەت زەنجىرى بەرپا قىلىشقا رىغبەتلەندۈرۈلۈپ،ئىشتەقسىماتى ئىلىمى ،ئۆز-ئارا مەنپەئەت يەتكۈزۈش ھالىتى دەسلەپكى قەدەمدە شەكىللەندۈرۈلدى.
